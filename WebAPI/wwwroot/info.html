Frequently Asked Questions (FAQ)

Q: How do I update my Telegram Bot Token?

A: Navigate to the "App Settings" page. Find the "TelegramPanel Settings" section and update the `TelegramPanel:BotToken` field. Remember to save changes. The application might need a restart for some core settings to take effect if dynamic reloading isn't fully supported for that specific setting yet.

Q: Where can I see if my database connection is working?

A: Go to the "Connection Test" page to test the database connection string you intend to use. The "System Status" page also shows the current live database connectivity status.

Q: How are sensitive settings like API keys stored?

A: When you save settings through the "App Settings" page, sensitive values are encrypted before being stored in the application's database. They are decrypted only when needed by the application. Environment variables can override these settings and are loaded directly by the application.

Basic Troubleshooting Tips

  • Bot not responding? Check the "System Status" page for Telegram API connectivity. Ensure your Bot Token is correct in "App Settings". Review the application logs on the "View Logs" page for any errors.
  • Can't log in to admin panel? Ensure the admin username and password configured in the application settings (or environment variables) are correct.
  • Settings not taking effect? Some critical settings might require an application restart after being changed. The system attempts to reload configurations dynamically where possible. Check logs for any errors during settings update or application startup.
  • Error messages on UI? Use browser developer tools (Console tab) to check for JavaScript errors. Toastr notifications at the top-right usually provide specific error details from API calls.

About This Admin Panel

This admin panel provides tools to configure, monitor, and manage the ForexTradingBot application. It allows direct interaction with application settings, diagnostic checks, and operational insights.

Key Features:

  • Secure login for administrators.
  • Dynamic application settings management with values stored securely in the database.
  • Connection testing for critical services (Database, Redis, Telegram Bot API).
  • Dashboard overview of key application statistics (users, signals).
  • System status monitoring for core connectivity and Hangfire jobs.
  • Log viewing and download capabilities.
